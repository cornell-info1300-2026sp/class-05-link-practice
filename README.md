# INFO 1300 (SP26) - Activity - Link Practice

In this activity we will practice adding a nav links to a project.

This is a practice activity - not for credit. Feel free to expand as inspired.

Open this repository as a Codespace on GitHub.
Start the web server by clicking "Go Live" in the bottom right-hand corner.

## Instructions 

- This activity is a continuation of our Ithaca Zoo example site.
  - Review the project folder.  Note there are 2 `.html` files. 
  - Review both `.html` files to see what we are starting with. 
  - Review the `styles/site.css` to see what we are starting with.

- View `index.html` in your browser using `Go Live` in the bottom right-hand corner.

- Add a navigation menu that links the 2 pages together
